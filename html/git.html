Git

  • git init - команда для создания нового локального репозитория Git.
  • git clone - команда для создания локальной копии удаленного репозитория Git.
  • git status - команда для получения информации о текущем состоянии вашего рабочего пространства Git.
  • git diff - команда для просмотра различий между версиями файлов в репозитории.
  • git push - команда для отправки изменений из вашего локального репозитория в удаленный репозиторий Git.
